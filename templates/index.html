How does it work?

Clicking 'Add' will tell the web app to start a new task, then display the progress page. In the code, you'll see that starting a new task is as simple as pushing some data onto a Redis list. The app is then free to complete the web request while the external worker.py process does all the work. Meanwhile the processing page, using AJAX, will poll the server until it responds with 'ready=true', to which it redirects you to the results page.
